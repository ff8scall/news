# 프로젝트 개요: Antigravity News

## 프로젝트 설명

**"Lego-Sia AI Tech Intelligence"**는 AI, 하드웨어, 게임 기술, 가이드에 초점을 맞춘 Hugo 기반 다국어 뉴스 플랫폼입니다. 이 프로젝트는 자동화된 콘텐츠 생성 및 큐레이션 시스템을 사용하여 한국어와 영어로 기술 인텔리전스를 제공합니다.

## 주요 기능
- **다국어 지원**: 한국어(주요) 및 영어 콘텐츠
- **자동화된 콘텐츠 파이프라인**: 뉴스 수집, AI 기반 작성, 콘텐츠 관리를 위한 Python 기반 자동화
- **4대 핵심 카테고리**: 
  - AI 모델 및 도구
  - GPU 및 하드웨어
  - 게임 기술 및 최적화
  - 전략 가이드
- **현대적 UI**: 색상 코드화된 카테고리 시스템과 반응형 디자인

## 아키텍처

### 프론트엔드 (Hugo 정적 사이트)
- **프레임워크**: Hugo (정적 사이트 생성기)
- **배포**: Netlify/Vercel 준비 완료
- **스타일링**: Tailwind 영감의 컴포넌트를 사용한 커스텀 CSS
- **콘텐츠 구조**: 
  - `content/ko/` - 한국어 콘텐츠
  - `content/en/` - 영어 콘텐츠
  - 클러스터 및 카테고리로 구성됨

### 백엔드 (자동화 시스템)
- **언어**: Python
- **핵심 구성요소**:
  - `news_harvester.py` - 뉴스 수집 및 처리
  - `ai_news_editor.py` - AI 기반 콘텐츠 편집
  - `ai_writer.py` - 콘텐츠 생성
  - `night_shift.py` - 자동화된 워크플로우 관리
  - `telegram_notifier.py` - 알림 시스템

### 콘텐츠 카테고리
| 메인 카테고리 | 세부 카테고리 | 색상 코드 |
|---------------|----------------|------------|
| AI | AI 모델·트렌드, AI 도구·사용법 | 파란색 (#3B82F6) |
| 하드웨어 | GPU·반도체, AI PC & 로보틱스 | 녹색 (#10B981) |
| 게임 테크 | 게임 최적화·엔진, AI 게임 기술 | 보라색 (#8B5CF6) |
| 가이드 | 실용 튜토리얼, 성능 비교 | 주황색 (#F59E0B) |

## 기술 스택
- **정적 사이트 생성기**: Hugo
- **백엔드 자동화**: Python 3.x
- **AI 모델**: Gemma 4, Google Gemini API
- **데이터베이스**: 뉴스 기록을 위한 SQLite
- **배포**: Netlify, Vercel
- **알림**: Telegram 봇

## 개발 상태
- **현재 버전**: V0 (Genesis)
- **상태**: 자동화된 콘텐츠 파이프라인으로 활발한 개발 중
- **최근 업데이트**: 
  - 분류 체계 재구성
  - 색상 코드화된 카테고리로 UI/UX 개선
  - 자동화된 뉴스 수집 및 AI 기반 콘텐츠 생성

## 주요 파일
- `hugo.toml` - Hugo 설정 및 사이트 구조
- `PROJECT_BRAIN.md` - 프로젝트 문서화 및 로드맵

- `automation/` - 완전한 Python 자동화 시스템
